    py-ds-academy¶

    Data structures implemented from scratch for learning and experimentation

    üéØ About¶

    py-ds-academy is a comprehensive collection of classic data structures implemented from scratch in Python. The goal is learning + correctness (with tests), not squeezing out every last micro-optimization.

    This project serves as both a learning resource and a reference implementation for common data structures used in computer science and software engineering.

    ‚ú® Features¶

    • üß± Core Data Structures - Stacks, queues, linked lists, trees, and heaps
    • üìù Type Hints - Full type annotations for better code clarity
    • üß™ Comprehensive Tests - TDD-first approach with pytest
    • üìö Well Documented - Clear APIs with complexity notes
    • üéì Educational - Perfect for learning data structures

    üöÄ Quick Start¶

    # Install dependencies
uv sync

# Run tests
uv run pytest

# Try it out
uv run python

    >>> from py_ds import Stack, Queue, MinHeap, BinarySearchTree

>>> # Stack example
>>> s = Stack([1, 2, 3])
>>> s.pop()
3

>>> # Queue example
>>> q = Queue([1, 2, 3])
>>> q.dequeue()
1

>>> # Heap example
>>> h = MinHeap([3, 1, 4, 1, 5])
>>> h.pop()
1
